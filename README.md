# heli ☀

Gamma brightness control for Hyprland. Lives in your terminal, saves your setting across reboots.

## install

```bash
curl -fsSL https://raw.githubusercontent.com/YusifKhalilov/heli/main/install.sh | bash
```

That's it. Type `heli` to open.

## controls

| key | action |
|-----|--------|
| `←` `→` or `h` `l` | adjust brightness |
| `r` | reset to normal |
| `q` | quit |

## requirements

- Hyprland
- `hyprsunset`
- Python 3 + `textual`
